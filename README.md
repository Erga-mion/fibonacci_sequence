# フィボナッチ数列

## 問題

フィボナッチ数列は以下の漸化式で与えられる。  
f(0) = 0  
f(1) = 1  
f(n+2) = f(n) + f(n+1) (n>=0)  

第n項の値を算出するプログラムを作成せよ。
